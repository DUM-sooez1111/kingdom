# CROWNVALE: 왕국 건설 — 브라우저판

외부 라이브러리나 설치가 필요 없는 단일 페이지 3D 건설 게임입니다.

## 실행

`index.html`을 Chrome, Edge, Firefox 등의 최신 브라우저에서 여세요.

## 조작

- 오른쪽 **건설**에서 건물을 선택한 뒤 월드의 소유 토지를 클릭해 배치
- `Q` / `E`: 카메라 회전
- 마우스 휠: 확대/축소
- **영토**: 잠긴 토지를 골드로 구매
- **노동자**: 세금 수집자를 고용하거나 왕실 자동 수금을 해금
- **인기도**: 편의시설·놀이기구·장식을 설치해 인기도를 올리고, 1점마다 모든 세금 수입 +0.5%

진행 상태는 해당 브라우저의 LocalStorage에 자동 저장됩니다. 상단의 저장 버튼은 즉시 저장합니다.

## 온라인 공개

정적 파일만으로 동작하므로 GitHub Pages, Netlify, Vercel, Cloudflare Pages 등 정적 호스팅에
`crownvale-browser` 폴더의 파일을 올리면 바로 공유 링크를 만들 수 있습니다.
